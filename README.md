## Built with:
- Typescript
- express
- PostgreSQL
- TypeORM
- multer
- cookie / cookie-parser
- bcrypt
- jwt
- class-transformer / class-validator

Live app link: comming soon :)



Steps to run this project:

1. Run `npm i` command
2. Setup database settings inside `ormconfig.json` file
3. Run `npm run dev` command
